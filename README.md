# 🛠 Парсер безмасляных компрессоров (ВсеИнструменты.ру)
> ⚠️ Проект предназначен исключительно для образовательных целей. Не аффилирован с ООО «ВсеИнструменты.ру».
> Перед использованием ознакомьтесь с полным текстом [DISCLAIMER.md](./DISCLAIMER.md)

Скрипт собирает данные о безмасляных компрессорах с категории:
🔗 [ВсеИнструменты.ру — Безмасляные компрессоры](https://www.vseinstrumenti.ru/category/bezmaslyanye-kompressory-394/)

### 📦 Собираемые данные

| Поле | Описание |
| :--- | :--- |
| **Наименование** | Полное название товара |
| **Бренд** | Производитель |
| **Цена** | Стоимость в рублях (без скидки) |
| **Цена_со_скидкой** | Акционная цена (если есть) |
| **Наличие** | Информация о доступности в магазинах |
| **Артикул** | Уникальный идентификатор товара |
| **Рейтинг** | Средняя оценка покупателей (0–5) |
| **Отзывы** | Количество отзывов |
| **Ссылка_на_изображение** | URL изображения товара |
| **Ссылка_на_товар** | Прямая ссылка на карточку товара |
| **Дата_парсинга** | Время сбора данных (ГГГГ-ММ-ДД ЧЧ:ММ:СС) |

### 🚀 Установка
```ruby
# Клонировать репозиторий
git clone <URL-репозитория>
cd compressor-parser

# Создать виртуальное окружение (рекомендуется)
python -m venv venv

# Активировать окружение
# Linux/macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt
```
### ▶️ Запуск
```ruby
python parser.py
```
> ⏱ При первом запуске скрипт ожидает 60 секунд для ручного прохождения капчи.
Результаты сохраняются в `data/vsi_perfect_result_YYYYMMDD_HHMMSS.xlsx`

### ⚙️ Настройки
Основные параметры находятся в функции `parse_pro_final():`
```ruby
WAIT_TIME = 60          # Время ожидания загрузки страницы (сек)
SCROLL_STEPS = 3        # Количество шагов прокрутки для lazy-load
TARGET_URL = "..."      # Целевая категория для парсинга
```
Для изменения параметров отредактируйте соответствующие строки в `parser.py.`

### 🛡️ Соответствие robots.txt
Скрипт разработан с учётом правил [robots.txt](https://www.vseinstrumenti.ru/robots.txt?spm=a2ty_o01.29997173.0.0.63eb55fbYVq1md&file=robots.txt):

- ✅ Не обращается к запрещённым путям: /checkout/, /api-common/, /user/
- ✅ Не использует заблокированные User-Agent (работает через undetected_chromedriver)
- ✅ Не добавляет запрещённые параметры в URL: ?sort=, ?filters=

Рекомендации при масштабировании:

- Добавляйте случайные задержки между запросами
- Проверяйте актуальность [robots.txt](https://www.vseinstrumenti.ru/robots.txt?spm=a2ty_o01.29997173.0.0.63eb55fbYVq1md&file=robots.txt) перед запуском
- Ограничивайте частоту запросов при парсинге нескольких страниц

- Python 3.10+

### 🐛 Устранение неполадок

| Проблема | Решение |
| :--- | :--- |
| **Браузер не запускается** | Убедитесь, что установлен Chrome; проверьте права доступа |
| **Капча не проходит** | Увеличьте `WAIT_TIME`; пройдите капчу вручную в открывшемся окне |
| **Данные не собираются** | Проверьте актуальность селекторов в коде; сайт мог изменить вёрстку |
| **Ошибка установки зависимостей** | Обновите pip: `python -m pip install --upgrade pip` |
